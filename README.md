# 顧客課程紀錄系統 ｜ 伸動保健室 💵

一套專為「伸動保健室」打造的輕量化、純淨離線版顧客與課程管理系統。
本系統移除所有繁瑣的雲端後台設定，採用最穩定的 `localStorage` 瀏覽器內建儲存技術，並延續伸動系列一貫的「全面置中」與「等寬數字」美學。

## 🚀 功能特色

- **純離線極速體驗**：免登入、免對接 Firestore、完全沒有任何雲端資費或資料外洩風險。
- **學員管理五件套**：
  - 📥 **一鍵匯入 / 匯出 (CSV)**：支援 Excel 與 Google 試算表相容的資料格式，隨時做本地實體備份。
  - ➕ **快速買課與明細**：直覺登錄購課堂數、實收金額，並自動換算每堂平均單價。
  - 🔄 **雙向動態排序**：點擊欄位即可針對名稱、年齡、累積堂數、總消費、購買次數及平均單價進行升冪或降冪排序。
  - 🔍 **複合式即時篩選**：支援關鍵字搜尋，並可自訂「購課超過 X 堂」的漏斗篩選。
  - 🧮 **全自動數據看板**：即時計算總顧客數、系統總售出堂數、總營業額，以及篩選後的加總數據。
- **伸動專屬美學**：全面置中的表格佈局，搭配等寬數字（Tabular Numbers）字體，確保金額與堂數在對齊時絕不抖動。
